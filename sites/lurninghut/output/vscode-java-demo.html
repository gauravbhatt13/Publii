VSCODE Java demo

Gaurav Bhatt

The goal of any IDE should be to increase developer productivity and in Java world today Intellij and VSCode are two popular IDEs used by the developers. In this demo I am going to show how we can integrate Java language tools by installing necessary VSCode extensions and create a productive Java development environment.

Demo

Install Extension Pack

Open the VSCode IDE

Now search for the extension in the extensions tab by searching for 'java extension pack'. As shown below we will see as part of the pack there are 6 extensions being pulled which are helpful while working with any Java based project.

Create a simple Java project

Once you install the extension pack, open Command Palette using Shift+Command+A and type 'java'. You will see options like 'Create Java Project' and click on it.
